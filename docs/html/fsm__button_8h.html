enum FSM_BUTTON

States for the finite state machine for the button.

Enumerator
BUTTON_RELEASED 

Starting state, button hasn't been pressed

BUTTON_PRESSED_WAIT 

Button has been pressed, wait some time to remove false positives from mechanical noise

BUTTON_PRESSED 

Button is currently pressed

BUTTON_RELEASED_WAIT 

Button has been released, wait some time to filter false positives from mechanical noise
